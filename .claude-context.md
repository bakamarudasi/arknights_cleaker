# Claude Code - プロジェクト構造ガイド

## メインプロジェクト (Unity)
```
Assets/
├── Scripts/           # ← メインのC#コード（ここを主に読む）
│   ├── Main_core/     # コアシステム（マネージャー、データ定義）
│   ├── Main_UI/       # UI関連（各画面のコントローラー）
│   └── Editor/        # Unityエディタ拡張
├── Items/             # ScriptableObjectアセット
├── Resources/         # ランタイムロード用リソース
└── Scenes/            # シーンファイル
```

## 読み飛ばしてOKなディレクトリ

| ディレクトリ | 理由 |
|-------------|------|
| `tools/` | 外部ツール。ゲーム本体とは独立 |
| `Library/` | Unity自動生成（gitignore済み） |
| `Logs/` | ログファイル |
| `Temp/` | 一時ファイル |

## データ定義の場所
- **アイテム**: `Assets/Scripts/Main_core/Data/Definitions/ItemData.cs`
- **アップグレード**: `Assets/Scripts/Main_core/Data/Definitions/UpgradeData.cs`
- **ガチャ**: `Assets/Scripts/Main_UI/Gacha_/Script/GachaBannerData.cs`
- **株式**: `Assets/Scripts/Main_core/Systems/Market/Data/`
- **キャラクター**: `Assets/Scripts/Main_core/Data/Character/`

## マネージャーの場所
- `Assets/Scripts/Main_core/Systems/` 配下に各システムのマネージャー
