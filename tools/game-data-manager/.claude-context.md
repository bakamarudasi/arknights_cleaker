# Claude Code - このディレクトリについて

## 概要
このディレクトリ (`tools/game-data-manager/`) は **メインのUnityプロジェクトとは独立した管理ツール** です。

## 読み飛ばしてOK
- ゲームのバグ修正・機能追加には関係ありません
- Unity側のコードを調査する際は、このディレクトリは無視してください

## このツールの用途
- ゲームデータ（アイテム、ガチャ、アップグレード等）をWebブラウザで編集するツール
- React + FastAPI で構成
- Unity本体のコードには影響しない

## いつ読むべきか
- ユーザーが「管理ツール」「データマネージャー」について言及した場合
- `tools/` フォルダ内の修正を依頼された場合

## 関連するUnity側のファイル
- `Assets/Scripts/Editor/DataManager/GameDataManagerWindow.cs` - Unity連携用エディタ拡張
